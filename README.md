# Grand Archive of the Seelie Court

An avalonia-based application hosting several tools.

## Rule Cards

Provides a renderer and editor for several data types, and an export function to convert the data into standard sized playing cards. Includes a toggle to add a bleeding edge border to the exported cards, so they can be printed using professional card printing services.
